# Routa CLI (NPM)

This package provides the `routa` command and delegates execution to a
platform-specific binary dependency.

## Install

```bash
npm install -g routa
```

or one-off usage:

```bash
npx routa --help
```

## Usage

```bash
routa -p "Implement a feature"
routa agent list
routa agent create --name "Coder" --description "Rust coding specialist"
routa task list
```
